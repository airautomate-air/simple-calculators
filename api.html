Getting Started

The Simple Calculators API provides free access to unit conversion functionality through RESTful endpoints. All responses are in JSON format.

Base URL

https://api.simplecalculators.com (or your deployed domain)

Request Format

All endpoints use GET requests with query parameters:

GET /api/[converter-type]?value=[number]&from=[unit]&to=[unit]

Response Format

Successful responses return JSON:

{ "value": 25, "from": "celsius", "to": "fahrenheit", "result": 77 }

Error Responses

Errors return JSON with HTTP status codes:

{ "error": "Invalid unit: 'xyz'. Valid units are: celsius, fahrenheit, kelvin" }

Available Endpoints

🌡️ Temperature Converter

Convert between Celsius, Fahrenheit, and Kelvin

GET /api/temperature
Parameters:
value (required): Number to convert
from (required): celsius, fahrenheit, kelvin
to (required): celsius, fahrenheit, kelvin
Example:
/api/temperature?value=100&from=celsius&to=fahrenheit
Returns: {"value":100,"from":"celsius","to":"fahrenheit","result":212}

📏 Length Converter

Convert between metric and imperial length units

Usage Examples

JavaScript (Fetch API)

fetch('/api/temperature?value=25&from=celsius&to=fahrenheit') .then(response => response.json()) .then(data => { console.log(`${data.value}°${data.from.toUpperCase()} = ${data.result}°${data.to.toUpperCase()}`); });

Python (Requests)

import requests response = requests.get('https://api.simplecalculators.com/api/length', params={'value': 5, 'from': 'ft', 'to': 'm'}) data = response.json() print(f"{data['value']} {data['from']} = {data['result']} {data['to']}")

cURL

curl "https://api.simplecalculators.com/api/weight?value=10&from=lb&to=kg"
