# Simple Calculators & Converters

A free, instant-access collection of calculators and unit converters for everyday use.

## Features
- Temperature converter (Celsius, Fahrenheit, Kelvin)
- Length, weight, volume converters
- Ingredient weight↔volume converter (coming soon)
- Minimalistic, clean, futuristic design
- Works offline, no tracking

## Development
This is a static site hosted on GitHub Pages.

To run locally, simply open `index.html` in a browser.
